Cara menginstal ZipForce
------------------------

Untuk menginstal ZipForce sangatlah mudah Anda tinggal Copy Paste code dibawah ini:

Code
----
1. apt-get update -y
2. apt-get install python3 -y
3. apt-get install python3-pip -y
4. apt-get install python3.11-venv -y
5. python3 -m venv /root/ZipForce/modules
6. source /root/ZipForce/modules/bin/activate
7. pip3 install -r /root/ZipForce/requirements.txt
